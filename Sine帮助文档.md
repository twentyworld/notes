**1. 如何pull repo到本地**

创建一个专用文件夹，然后
把note从github上拉下来的指令：

```shell
git clone https://github.com/twentyworld/notes.git
```
---

**2. 在编辑器中编辑文件**
具体工具请查看不同的文件查看详情。
或者浏览[这里][1]
---

**3. 把改动push到github**

terminal/终端/cmd 中cd 到项目的根目录，在本项目中，应该是-> .../notes/

```command
git add .
git commit -m " commit message"
git push

```

and do as youlike.
